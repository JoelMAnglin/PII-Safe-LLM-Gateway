# Recruiter Demo Script

Use this script during an interview or portfolio walkthrough.

## 30-Second Pitch

```text
I built a privacy gateway for LLM workflows. It ingests text, JSON, or CSV data, detects possible identifying attributes, de-identifies them, creates an audit report, and produces a sanitized prompt preview. The goal is to reduce accidental exposure of customer, employee, or incident data before someone sends it to an AI tool.
```

## Demo Steps

1. Open the browser demo.
2. Load the customer support sample.
3. Point out raw PII in the input.
4. Click sanitize.
5. Show the sanitized output.
6. Show the privacy report.
7. Show the safe prompt preview.
8. Run the CLI command.
9. Run the tests.
