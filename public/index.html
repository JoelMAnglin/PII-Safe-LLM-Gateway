  2. Choose a mode.Replace is best for LLM safety. Mask and hash are useful for audits.
  3. Sanitize.The app detects likely PII and secrets, then removes or tokenizes them.
  4. Review the report.Confirm which attributes were found before sending the prompt onward.
  5. Copy safe prompt.Only the sanitized prompt should go to an LLM.
